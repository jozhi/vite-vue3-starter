# Vite 2.x + Vue 3.x + TypeScript Starter

## 快速开始

### 获取项目

#### 使用 Git

```sh
git clone https://github.com/XPoet/vite-vue3-starter.git
```

#### 使用 NPM

```sh
npm install vite-vue-ts-cli -g

vite-vue-ts-create myapp
```

### 安装依赖

```sh
npm install
# or
yarn add
```

### 启动项目

```sh
npm run dev
```

### 项目打包

```sh
npm run build
```
